## Install

To install the chart with release name `apisix`:

```shell
helm repo add apisix https://charts.apiseven.com
helm repo update
helm install apisix apisix/apisix
```

## Uninstall

To uninstall/delete the `apisix` release:

```shell
helm uninstall apisix
```
